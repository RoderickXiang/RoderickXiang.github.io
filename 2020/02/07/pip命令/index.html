pip命令

发表于 2020-02-07

安装包

pip install <包名>

查看已安装的包

pip list

查看可升级的的包

pip list -o

升级

pip install -U <包名>

卸载包

pip uninstall <包名>

指定单次安装源

pip install <包名> -i https://mirrors.aliyun.com/pypi/simple

显示包所在的位置

pip show <包名>
